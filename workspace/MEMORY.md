## 已知注意事项

- **parent_wiki_token 不能传 space_id**：飞书空间ID（纯数字如 `7618158120166034630`）不是有效的 wiki token，v0.8.8 已修复但仍需注意 LLM 传参
- **响应延迟极高**：所有交互延迟均超过 70 秒，平均约 2-3 分钟。原因：LLM 调用平均 66 秒 + 工具调用链积累。用户已明确抱怨过，这是高优先级问题
- **admin-http 端口冲突**：8080 端口重复占用导致 admin 线程持续崩溃，需在重启前 kill 端口
- **工具调用层级深**：每次 LLM 调用平均携带 ~50 个工具（应该是动态注入后的正常值，但仍偏重）
- **用户无法看到回复**：历史上发生过一次用户完全看不到回复的问题（output 被过滤或清空），已修复
