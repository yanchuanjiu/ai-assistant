1. **大模型产品知识库** — 美妆护肤产品知识梳理，服务营销培训/美容顾问/电商运营/客服

## 已知 API 坑点（操作前必读）

- **parent_wiki_token 不能传 space_id**：飞书空间ID（纯数字如 `7618158120166034630`）不是有效 wiki token，`_is_space_level_token()` 会兜底，但 LLM 传参时仍需注意
- **tenant_access_token 不支持 wiki spaces API**：`/wiki/v2/spaces` 系列会报 131006 权限错误，这是需要管理员配置的问题（配置 `FEISHU_WIKI_ROOT_NODES` 或添加应用为空间成员），Agent 无法自主解决
- **代码生效需重启**：每次 commit 后必须重启服务，否则 Python 内存中仍运行旧代码

3. **项目进展梳理**：钉钉历史会议提取 → 整理成周报写入飞书 `05_状态周报`
