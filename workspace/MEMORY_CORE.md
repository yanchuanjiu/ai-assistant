## 已知 API 坑点（操作前必读）

- **parent_wiki_token 不能传 space_id**：飞书空间ID（纯数字如 `7618158120166034630`）不是有效 wiki token，`_is_space_level_token()` 会兜底，但 LLM 传参时仍需注意
- **token 续期自动处理**：access_token 过期由系统（`feishu_call` + `get_user_access_token`）自动续期（v2 端点），遇到 token/权限错误**不要调用 `feishu_oauth_setup` 诊断**，系统会自动通知用户
- **131006 权限问题**：系统自动通过飞书 IM 通知用户（三选一解决：添加应用为空间成员 / 配置 FEISHU_WIKI_ROOT_NODES / 重新授权）；Agent 返回提示即可，不尝试自主绕过
- **代码生效需重启**：每次 commit 后必须重启服务，否则 Python 内存中仍运行旧代码
